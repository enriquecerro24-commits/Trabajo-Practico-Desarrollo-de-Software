# Trabajo-Practico-Desarrollo-de-Software
El sistema consiste en una aplicación web destinada a la gestión integral de un gimnasio de tamaño medio, permitiendo administrar usuarios, actividades, turnos y reservas, optimizando la organización de clases y el uso de recursos disponibles
## Objetivo
Desarrollar un sistema que centralice la gestión de turnos y actividades, mejorando la eficiencia operativa y la experiencia de los usuarios.

## Alcance
El sistema permitirá:
- Registro y gestión de usuarios
- Creación y administración de actividades
- Asignación de turnos
- Reserva de clases

## Actores del sistema
- Administrador
- Entrenador
- Cliente

## Funcionalidades principales
- Gestión de usuarios (alta, baja, modificación)
- Gestión de actividades
- Reserva de turnos
- Visualización de disponibilidad
- Administración de horarios
